# KeyBART
Biomedical keyword generation with Bart

This repository contains a fine-tuned version of the Bart model that generates high-level biomedical concepts from any given input text. You can load this model by using the hoggingFace package. This model is fine-tuned on 1 million PubMed abstracts and author's defined keywords.

The Model is accessible from this link : https://drive.google.com/drive/folders/1_EJyJH1Wo-KgjntsjncBsDSa1drYFaeU?usp=sharing
